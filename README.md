# Waktu Sholat App 🕌

App sederhana: buka → langsung redirect ke Google result waktu sholat Mustika Jaya Bekasi.

---

## Cara Deploy ke GitHub Pages (sekali aja)

### 1. Buat akun GitHub
Kalau belum punya, daftar di [github.com](https://github.com).

### 2. Buat repository baru
- Klik tombol **+** → **New repository**
- Nama repo: `waktu-sholat` (bebas)
- Centang **"Add a README file"**
- Klik **Create repository**

### 3. Upload file-file ini
- Buka repo yang baru dibuat
- Klik **Add file** → **Upload files**
- Upload semua file dari folder ini:
  - `index.html`
  - `manifest.json`
  - `icon-192.png`
  - `icon-512.png`
- Klik **Commit changes**

### 4. Aktifkan GitHub Pages
- Buka tab **Settings** di repo kamu
- Scroll ke bagian **Pages** (di sidebar kiri)
- Di **Source**, pilih **Deploy from a branch**
- Branch: **main**, folder: **/ (root)**
- Klik **Save**

### 5. Tunggu ~1 menit
GitHub Pages akan kasih URL seperti:
`https://usernamekamu.github.io/waktu-sholat/`

---

## Install ke HP Android

1. Buka URL tadi di **Chrome**
2. Tap menu titik tiga (⋮) di pojok kanan atas
3. Pilih **"Add to Home screen"** / **"Tambahkan ke layar utama"**
4. Kasih nama, tap **Add**

Selesai! Sekarang ada icon app di home screen kamu. 🎉

---

## Install ke iPhone (iOS)

1. Buka URL di **Safari** (harus Safari, bukan Chrome)
2. Tap tombol **Share** (kotak dengan panah ke atas)
3. Pilih **"Add to Home Screen"**
4. Tap **Add**

---

Sekali tap → langsung muncul waktu sholat. Done.
